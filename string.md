# String

## includes

```js
'abc'.includes('bc'); // true
'abc'.includes('cb'); // false
```

## indexOf

```js
'abc'.indexOf('bc'); // 1
'abc'.indexOf('cx'); // -1
```

## padEnd

```js
'1'.padEnd(3, '*'); // '1**'
'100'.padEnd(3, '*'); // '100'
```

## padStart

```js
'1'.padStart(3, '0'); // '001'
'1000'.padStart(3, '0'); // '1000'
```
