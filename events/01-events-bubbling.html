Распространение событий в javascript

Всплытие событий

Если обработчики реагируют на фазу всплытия, то при клике на inner сначала сработает handler_inner, затем handler_outer. Это связано с тем, что при всплытии событие распространяется от внутреннего элемента к внешнему.

Если кликнуть на внешнем элементе outer, то обработчик handler_inner не сработает.

#inner

Код


var outer = document.getElementById('outer');
var inner = document.getElementById('inner');

var handler_outer = function () {
  alert('outer');
};

var handler_inner = function () {
  alert('inner');
};

//Обработчик сработает при всплытии события
outer.addEventListener('click', handler_outer, false);
inner.addEventListener('click', handler_inner, false);
