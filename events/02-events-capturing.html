Распространение событий в javascript

Фаза захвата

Если обработчики реагируют на фазу захвата, то при клике на inner сначала сработает handler_outer, затем handler_inner. Это связано с тем, что при захвате событие распространяется от внешнего элемента к внутреннему.

Если кликнуть на внешнем элементе outer, то обработчик handler_inner не сработает.

#inner

Код


var outer = document.getElementById('outer');
var inner = document.getElementById('inner');

var handler_outer = function () {
  alert('outer');
};

var handler_inner = function () {
  alert('inner');
};

//Обработчик сработает при захвате события
outer.addEventListener('click', handler_outer, true);
inner.addEventListener('click', handler_inner, true);
